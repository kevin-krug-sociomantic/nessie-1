Component Description
---------------------

A Nessie Accordion is used as the container of AccordionItem components

Example Usage
-------------
<Accordion>
	<AccordionItem headerText="Form"/>
		<TextInput label="your name" >
	</AccordionItem>
	<AccordionItem headerText="Form"/>
		<TextInput label="your name" >
	</AccordionItem>
</Accordion>
